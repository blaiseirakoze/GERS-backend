## Getting started

-   Clone the project

```
    > git clone <url>
```

-   Navigate to the project repository

```
    > cd nppa
```

-   Generate .env

```
    > copy .env.example .env
```

-   Install

```
    > yarn install or npm install
```

-   Run

```
    > yarn start or npm start
```
